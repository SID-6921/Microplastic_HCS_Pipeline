Results overview

This folder contains regenerated tables, figures, and appendices used in the manuscript draft.

- `results/tables/` — CSV tables used for manuscript tables.
- `results/figures/` — PNG figures used in the manuscript.
- `results/appendices/` — Draft appendices and metadata templates (authors to verify).

If you wish to reproduce the results, ensure `pipeline/features.py` is present and run `python scripts/build_all_results.py`.
