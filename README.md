# Microplastic HCS Pipeline

Production-ready repository for the MS2 high-content screening pilot on microplastic-associated cell-state classification.

## What this repo contains

- `pipeline/` - image loading, preprocessing, nucleus detection, feature extraction, and classification helpers.
- `scripts/` - the end-to-end rebuild script that generates tables and figures.
- `notebooks/` - the manuscript notebook used to present the final outputs.
- `results/` - publication artifacts generated by the rebuild pipeline.
- `docs/` - appendix material and author-facing confirmations.
- `tests/` - smoke tests that validate the core pipeline.

## Quick start

```powershell
python -m pip install -r requirements-dev.txt
pytest
```

If you want to regenerate the figures and tables:

```powershell
python scripts/build_all_results.py
```

If you need a faster validation run, set `TARGET_PER_CLASS` before launching the build:

```powershell
$env:TARGET_PER_CLASS = "20"
python scripts/build_all_results.py
```

## Notebook

Open `notebooks/MS2_Manuscript.ipynb` after the build completes to review the generated outputs. The notebook is written to load from the local `results/` directory.

## Reproducibility

- `requirements.txt` captures the runtime dependencies.
- `requirements-dev.txt` adds notebook execution and test tooling.
- `RELEASE_NOTES.md` summarizes the publishable release state.
- `LICENSE` is MIT for straightforward GitHub publication.

## Provenance

The repository is intentionally framed as a pilot feasibility package. The manuscript notebook and generated outputs should be treated as release artifacts for the MS2 submission, not as a final clinical product.
